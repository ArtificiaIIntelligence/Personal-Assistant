1. For using a pyttsx as speach synthesis is necessary to install it : on linux -sudo pip install pyttsx. 

2. The pyttsx is using espeak on linux(-sudo apt-get install espeak).

3. If you are using python 3.x, you will need to make changes in code. You can download code from GitHub and rewrite it (in .\Python\Python35-32\Lib\site-packages\pyttsx).
