The scripts were tested with python3.4. 
The geopy API must be installed first, for example using the pip installer.

The keyword recognition runs in python 2.7.
The wit API must be installed first, for example using the pip installer.
The website: https://wit.ai/docs/python/1.0.0/quickstart

The Festival Speech Synthesis System is used to Text to speech (prototype). python3.4.
The festival API must be installed first. 
Ex:
sudo apt-get install python3 python3-dev festival festival-dev festvox-kallpc16k
sudo pip3 install pyfestival
